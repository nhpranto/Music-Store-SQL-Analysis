# Music Store SQL Analysis
Overview

Analyzed a digital music store database using SQL to answer real business questions about customers, sales, employees, and music preferences across countries.

# Skills Used

SQL (Joins, Aggregations)

Window Functions (ROW_NUMBER)

Common Table Expressions (CTEs)

Data Analysis & Business Insights

# Key Questions Answered

Who is the senior-most employee?

Which countries generate the most invoices?

Who are the top-spending customers?

What is the most popular music genre by country?

Who is the highest-paying customer in each country (including ties)?

# Highlights

Ranked customers and genres by country using window functions

Combined multiple tables with efficient joins

Used CTEs to simplify complex queries

Focused on insights that support sales and marketing decisions

# Business Value

Identifies high-value customers

Reveals top-performing regions

Supports region-specific marketing strategies
